## Build with CMake

```bash
cmake -S . -B build
cmake --build build
```

Executable output:

- `build/bin/OpenGL.exe` (Windows)

GLFW is fetched and built automatically by CMake, so you don't need to open the Visual Studio `.sln` directly.
